The source code and jar file of share-memory word count are in share-memory wordcount folder.

Share-memory sort source code and jar file is in share-memory sort folder.


Compile command:
javac Share-memory_WordCount.java



Usage of wordcountJ.jar (share-memory):

Open terminal, access the directory, type the following command to run the share-memory word count:

java -jar WordCountJ.jar THREADS OUTPUT [FILES...]




For sort program:

Compile command:
javac Share-memory_Sort.java

Usage of wordsort.jar:
java -jar wordsort.jar input output
